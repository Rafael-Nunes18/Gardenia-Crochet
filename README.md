# Gardenia Crochet

Gardenia Crochet é um site dedicado à venda de produtos de crochê feitos à mão. Nossa missão é oferecer peças exclusivas, feitas com carinho e atenção aos detalhes, trazendo beleza e aconchego para o seu dia a dia.

## Sobre o Projeto

Este repositório contém o código-fonte do site Gardenia Crochet. O objetivo da plataforma é facilitar a exposição e venda dos nossos produtos, proporcionando uma experiência agradável e intuitiva para nossos clientes.

## Funcionalidades

- Catálogo de produtos de crochê, com fotos e descrições detalhadas
- Carrinho de compras para facilitar pedidos
- Sistema de contato para dúvidas e encomendas personalizadas
- Página sobre a marca e o processo artesanal

## Tecnologias Utilizadas

- **Front-End:** Figma (prototipação), HTML, CSS, JavaScript
- **Back-End:** C#, ASP.NET MVC
- **Banco de Dados:** Microsoft SQL Server 2022 (modelagem via Draw.io)
- **Deploy:** AWS EC2
